
# 戦闘破壊学園ダンゲロス オンライン対戦支援ツール

ウォーシミュレーションゲーム「戦闘破壊学園ダンゲロス」のオンライン対戦を円滑に進めるためのブラウザベースの支援ツールです。

## 🚀 使い方 (オンライン同期)

### 同期の手順
1. プレイヤー全員が同じ **「部屋名」** を入力して入室します。
2. 最初にその部屋名で入った人が「ホスト」となり、以降の参加者はホストの盤面データを自動的に受け取ります。
3. 盤面での移動、チャット、ステータス変更がリアルタイムに全プレイヤーに同期されます。

※このツールは **PeerJS (P2P通信)** を使用しています。サーバーを介さないため、動作は無料かつリアルタイムですが、通信環境によっては接続に数秒かかる場合があります。

## 🛠 主な機能

- **オンライン同期**: 同じ部屋名のプレイヤー間で盤面、チャット、ステータスをリアルタイム共有。
- **5x7 グリッド盤面**: ユニットや壁を自由にドラッグ＆ドロップで配置・移動可能。
- **ユニット管理**: キャラクターの作成、ステータス（攻/防/体/精/FS）のリアルタイム編集、生存/死亡/能力休み状態の管理。
- **ダイスロール機能**: チャット欄で `1d100` などのコマンドを使用してダイスを振ることができ、結果は全プレイヤーに共有されます。
- **通常攻撃計算**: `15 => 5` のような形式で攻撃力と防御力を入力すると、命中率とダメージを自動計算。
- **陣営管理**: 陣営ごとにDP（ダメージポイント）や生存数を管理。

## 📦 使用技術

- **Frontend**: React (v19), Tailwind CSS, Lucide React (Icons)
- **State Management**: Zustand
- **P2P Communication**: PeerJS
- **Drag & Drop**: dnd-kit
- **Build Tool**: Vite
